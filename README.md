# Credora

**Credora** is a CompTIA study and practice website built to help students review for **A+**, **Network+**, and **Security+** in one clean, organized place.

The site separates each certification into its own section so users do not get mixed questions or notes from the wrong exam. Each cert page includes detailed notes, practice options, filters, and review tools.

## Features

- Separate study sections for:
  - CompTIA A+
  - CompTIA Network+
  - CompTIA Security+
- Organized certification landing cards
- Detailed study notes for each exam
- Practice mode for each certification
- Question filters by:
  - Difficulty
  - Domain
  - Timed mode
  - Session size
- Individual question review buttons
- Dark and light mode settings
- Adjustable question font size
- “Back to Top” buttons on note pages
- Clean homepage with recommended resources
- Browser tab icon/favicon
- Footer notice with copyright and affiliation disclaimer

## Purpose

Credora was created to make CompTIA studying feel less scattered. Instead of mixing A+, Network+, and Security+ content together, the site lets users choose the certification they are studying for and stay focused inside that section.

The goal is to make certification prep easier, cleaner, and more useful for students who want notes and practice questions in one place.

## Certifications Covered

### CompTIA A+

Covers hardware, operating systems, mobile devices, networking basics, security fundamentals, troubleshooting, and support skills.

### CompTIA Network+

Covers networking concepts, IP addressing, ports, protocols, the OSI model, routing, switching, wireless networking, network security, troubleshooting, and network operations.

### CompTIA Security+

Covers security concepts, threats, vulnerabilities, cryptography, identity and access management, incident response, risk, governance, and security operations.

## How to Use

1. Open the website.
2. Choose the certification you want to study.
3. Read through the notes for that certification.
4. Click the practice button when you are ready.
5. Use the filters to choose difficulty, domain, timing, and session size.
6. Start practice or review individual questions.
